## About

This is a web page, which is mainly based on javascript with handlebars and pages created on hbs. With the main purpose to view the possible activities
---

## Project Structure
Controllers => each page javascript file, for the activity special one for all
Models => json file as a storage and app-store.js and json-store.js which are mandatory to json file work
Public => images folder, with css file in it
Utils => logger.js is in here 
Dependencies => package-lock.json, package.json

Port, Functionality => routes.js and server.js
